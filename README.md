# Rule-Based Chatbot

A simple, extensible rule-based chatbot built in Python. This chatbot uses pattern matching to respond to user inputs based on predefined rules.

## Features

- 🔍 Pattern matching using regex
- 🎲 Random response selection for variety
- 📝 JSON-based rule configuration
- 🔧 Extensible - easily add new rules
- 💬 Interactive command-line interface
- 💾 Save rules to file
